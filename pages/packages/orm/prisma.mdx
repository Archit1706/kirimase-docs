# Prisma
Prisma is an open source Node.js and TypeScript ORM with an intuitive data model, automated migrations, type-safety, and auto-completion.

[Documentation](https://www.prisma.io/docs/getting-started/setup-prisma/add-to-existing-project/relational-databases-typescript-postgresql)

  * @t3-oss/env-nextjs
* Developer
  * prisma
  * zod-prisma
